# Submission-analisis-data E-Commerce 🛍️

Dashboard ini dibuat sebagai bagian dari proyek akhir analisis data untuk Dicoding. Dashboard menampilkan wawasan tentang kategori produk terlaris, metode pembayaran, dan analisis RFM.

## Setup Environment - Anaconda

conda create --name main-ds python=3.9

conda activate main-ds

pip install -r requirements.txt

## Setup Environment - Shell/Terminal

mkdir proyek_analisis_data

cd proyek_analisis_data

pipenv install

pipenv shell

pip install -r requirements.txt

## Run streamlit app
Pastikan Anda berada di direktori utama proyek, lalu jalankan:

streamlit run dashboard/dashboard.py
